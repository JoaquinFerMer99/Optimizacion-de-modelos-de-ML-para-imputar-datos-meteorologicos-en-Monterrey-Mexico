# Optimización de modelos de ML para imputar datos meteorológicos en Monterrey, México
Este repositorio presenta el desarrollo, implementación y evaluación de modelos de aprendizaje automático (ML) aplicados a la imputación de series climatológicas incompletas.
El proyecto aborda el problema de los datos faltantes en registros de precipitación provenientes de estaciones meteorológicas, una limitación común en la modelación hidrometeorológica.

Se comparan modelos tradicionales con métodos avanzados de ML, como Random Forest y Gradient Boost, bajo un protocolo de validación espaciotemporal diseñado para evitar fuga de información y medir desempeño por estación y patrón de ausencia.
