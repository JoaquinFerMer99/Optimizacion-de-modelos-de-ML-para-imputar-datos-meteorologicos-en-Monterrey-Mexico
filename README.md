# Optimizaci-n-de-modelos-de-ML-para-imputar-datos-meteorol-gicos-en-Monterrey-M-xico
Proyecto de tesis de maestría orientado a la imputación de datos meteorológicos mediante algoritmos de aprendizaje automático. Evalúa y compara modelos y arquitecturas con atención espaciotemporal, bajo validación espaciotemporal y métricas de error estándar en climatología.
